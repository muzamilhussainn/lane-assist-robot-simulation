# Lane Assist Robot Simulation

This project is a lane-following robot simulation built using ROS Noetic, Gazebo, OpenCV, and feedback control concepts.

## Project Overview

The goal of this project was to simulate a robot that can detect a lane and adjust its movement based on the position error. The system follows a modular ROS publisher-subscriber structure, where perception and control are handled through separate components.

## Tools and Technologies

* ROS Noetic
* Gazebo
* Python
* OpenCV
* Feedback Control
* Publisher-Subscriber Communication

## Key Features

* Lane detection using computer vision
* Real-time position error calculation
* Steering correction using proportional control
* Signal smoothing for stable robot movement
* Modular ROS-based architecture
